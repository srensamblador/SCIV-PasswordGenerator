About

Super Castlevania IV is a 1991 game developed by Konami and the fourth entry in the mainline Castlevania series. The game didn't have a save system, and instead, utilized passwords to keep the player's progress.

At the start of the game, the player must input his name. During the course of the playthrough, if the player runs out of lives, the game gives him the option to continue or see the password from the last checkpoint he reached.

Said password is not a string of text, but a grid of 4x4 symbols (axe, holy water, heart or blank) generated from the name of the player, the checkpoint and the game mode (first playthrough or second quest). The player would henceforth be able to input his name and the password from the start screen to continue from that point.

This website generates all of the passwords belonging to a given player name. Note that the player name must be composed of up to 8 alphanumeric characters, no symbols whatsoever.
